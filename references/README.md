# Reference Genome Files

Large reference FASTA, GFF, and Bowtie2 index files are not stored in this repository.
They can be regenerated from the public RefSeq assembly used in the project:

- Species: *Cyprinus carpio*
- Assembly: `GCF_018340385.1_ASM1834038v1`
- Chromosome analyzed downstream: A20 / `NC_056591.1`

Example commands are provided in `REPRODUCING.md`.
